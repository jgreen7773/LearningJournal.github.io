Scripts, Expressions, Operators, and Functions!

  • A script is a series of instructions
    1. Define the goal
    2. Design the script
    3. Code each step
  • An expression is simply a piece of code that defines or returns a variable
    • Expressions rely on things called operators
      1. Example : var color = 'beige';
      2. Example : var area = 3 * 2;
  • Operators allow programmers to create a single value from one or more values
    • List of Arithmetic Operators
      1. + is an addition operator: it adds one value to another
      2. - is a subraction operator: it subracts values from others
      3. / is a division operator: it divides two values
      4. * is a multiplication operator: it multiples integers
      5. ++ is an increment operator: it is used in FOR loops; it adds one value to the current number
      6. -- is a decrement operator: it is also used in loops; it subracts one value from the current number
      7. % is the modulus operator: it divides two values and returns the remainder
    • Using Arithmetic Operators
      1. var subtotal = (13 + 1) * 5; = subtotal is 70
      2. var shipping = 0.5 * (13 + 1); = shipping is 7
      3. var total = subtotal + shipping; = total is 77

    • Using String Operators

  • Functions let you group a series of statements together to perform a specific task.
    • A "Function Declaration" cosists of:
      1. function keyword (which is 'function')
      2. function name (what ever you want to name the function)
      3. code block (the curly braces that embrace the code)
      4. statements, parameters, commands, and more functions (the defining lines of code that direct the full course of the function)
    • Calling a Function consists of a function name, paranthesis (to define a condition), and a semi-colon (to close the statement)
      • If a function requires a condition: parameters are entered into (paranatheses) before statements are configured.
